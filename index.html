Guest Book

This project is a fully deployed guestbook web application. It was built with the MEAN stack, an acronym for MongoDB, Express, Angular, and Node. It demonstrates full create, read, update, and delete database access implemented with the Mongoose package. Image upload was added with the Multer package. Authentification was added with Bcrypt and JSON Web Token. The project is deployed on AWS Beanstalk and S3 servers.
